## 5.2 Load HR Policy PDF(s) and split into token-aware chunks

Chunk size and chunk overlap directly control how much context the model sees at query time, because retrieval happens at the chunk level.
- More chunks (small chunk_size, e.g. 200 tokens) = more fragmentation = weaker context continuity.
   - A single logical idea may be split across 3–5 chunks. Retrieval may return only one of them. The model sees incomplete information. Answers become shallow or hallucinated because the model lacks the full context.
   - This is the classic “RAG fragmentation problem"

- Fewer chunks (large chunk_size, e.g. 2000 tokens) = more coherent context but higher token cost.
   - Each chunk contains full ideas, sections, or arguments. Retrieval returns richer context. The model answers more accurately.
   - BUT: Embedding cost increases. Vector DB size increases. Retrieval becomes slower. You risk exceeding the model’s context window when combining multiple chunks.
   - This is the “large chunk cost problem"

Chunk overlap preserves continuity (e.g., 128 tokens) ensures:
- If an important sentence sits at the boundary between chunks. It appears in both chunks. Retrieval is more likely to return at least one chunk containing the full idea. Overlap is a continuity insurance policy.
- But too much overlap: Increases total number of chunks. Increases embedding cost. Adds redundancy.

```python
from pypdf import PdfReader

# Detect whether the path is a folder (multiple PDFs) or a single file
is_folder = os.path.isdir(PDF_PATH_OR_FOLDER)

# Token-aware text splitter (uses the OpenAI/GPT tokenizer encoding for accurate token counts)
# Adjust chunk_size / chunk_overlap to suit your tradeoff between context and number of chunks
CHUNK_SIZE = 800
CHUNK_OVERLAP = 128

if is_folder:
    # Folder case: use PyPDFDirectoryLoader to load all PDFs and split using the splitter
    text_splitter = RecursiveCharacterTextSplitter.from_tiktoken_encoder(
        encoding_name="cl100k_base",
        chunk_size=CHUNK_SIZE,
        chunk_overlap=CHUNK_OVERLAP
    )

    pdf_loader = PyPDFDirectoryLoader(PDF_PATH_OR_FOLDER)
    docs = pdf_loader.load_and_split(text_splitter)

else:
    # Single file case: extract text using pypdf then split with the same token-aware splitter
    reader = PdfReader(PDF_PATH_OR_FOLDER)
    pages = []
    for p in reader.pages:
        pages.append(p.extract_text() or "")

    full_text = "\n\n".join(pages)

    text_splitter = RecursiveCharacterTextSplitter.from_tiktoken_encoder(
        encoding_name="cl100k_base",
        chunk_size=CHUNK_SIZE,
        chunk_overlap=CHUNK_OVERLAP
    )

    # create_documents returns a list of langchain Document objects
    docs = text_splitter.create_documents([full_text])
```
